# DHIMIX AI Changelog

All notable changes to DHIMIX AI are documented here.

## v1.0.0 — May 2026 — Free Tier Launch

### Platform
- Live AI music recommendations with full explainability
- Jamendo open music catalog integration
- Cloudflare R2 Cloud Showcase streaming
- 3 AI scoring modes — Genre-First · Mood-First · Energy-Focused
- Confidence scoring — HIGH / MEDIUM / LOW on every result
- Why[] reasoning panel per recommendation

### User Intelligence
- Music DNA preference vector — cross-session learning
- Favorites with localStorage persistence
- Listening history — last 100 tracks

### Platform Features
- License Center — per-track licensing status
- Analytics dashboard — live playback telemetry
- Library page — Cloud Showcase browsable grid
- Source filter — All Sources · Jamendo Only · Cloud Showcase Only
- AI Insights dashboard

### Infrastructure
- Vercel global edge deployment
- Cloudflare R2 signed URL streaming
- FastAPI backend
- R2 fallback mode when Jamendo is unavailable
